The lammps_src code contains custom lammps reaction package source codes. The lammps version used for this project was lammps-15Jun2023. Lammps must be built with these replaces reaction packages in the src/REACTION folder.

Each of the folders in model_scripts has the lammps inputs and instructions for running the simulation experiments: Stretch (Fig3), 7s_Perturbations (Fig4), and Ablations (FigS3). The GaussianSpread data processing was used for the analysis in Figs 3&4. The folder model_example has a basic annotated script of the collagen model. 

Figures has plotting scripts and data for the various figures in the paper.
